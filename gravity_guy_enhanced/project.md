My term project for MECHTRON 2TA4 is a remake of Gravity Guy using an ultrasonic sensor. The game is an infinite runner where the objective is to run for as long as possible without crashing. There are obstacles on the top and bottom tracks, and the player must flip gravity to dodge them and continue running, while they speed up as their score increases. The sensor detects hand waves within 20 cm, and each wave triggers a gravity flip. There is a shift register connected to a series of LEDs that light up in order as the player's score goes up, and they all flash a few times on death.
